Building the source code

On Linux, extract the download tarball or get the source from Github:

$ git clone git://github.com/scalien/scaliendb.git scaliendb

Then enter the source directory, and issue the following commands:

$ cd scaliendb
$ make
$ make pythonlib # optional
$ make javalib # optional

On Windows, use the supplied Visual Studio project files.

Development branch

To download the latest development branch:

$ git clone git://github.com/scalien/scaliendb.git scaliendb

Once the server is up running, connect to http://localhost:8080/webadmin to see the Web Management Console.

Don't forget to check out the documentation.
