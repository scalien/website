What is ScalienDB?

ScalienDB unites the strengths of SQL databases with the strengths
of NoSQL databases. ScalienDB is designed from the ground up to be a
highly available and scalable Transactional NoSQL database for the Enterprise.

ScalienDB is available for free under an open source license. We also offer Commercial Licensing and other Enterprise services such as Support, Training and Proof of Concepts. For more information please visit our Services page.

Let's take a look at the key features of ScalienDB and how they benefit you or download the full ScalienDB whitepaper.

Transactions

ScalienDB in the only open source NoSQL database to support fully ACID transactions, making it a “system of record” primary database for the Enterprise.

Scalable

ScalienDB is a horizontally scalable distributed database. To store more data you add more commodity servers instead of buying bigger, more expensive mainframe-like servers.

Schemaless

ScalienDB is a schemaless key-value database with high control over data layout. This gives maximum flexibility and fits in nicely with continuous integration and agile methodologies.

No Single Point of Failure

You can run ScalienDB at 3x or higher replication for fault tolerance. Unlike other NoSQL databases, ScalienDB can tolerate failures up to a single remaining data server.

Immediate Consistency

ScalienDB uses the Paxos algorithm for replication. This means that ScalienDB is immediately consistent unlike other NoSQL databases that are only eventually consistent.

No Downtime Migration

After major application changes, data has to be transformed from the old representation to the new one. With ScalienDB this can be performed on-demand, without downtime.
