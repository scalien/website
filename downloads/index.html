Notes

  • Check the Release log for changes
  • The downloads include ScalienDB, the Standard Management Console and sample configurations.
  • We recommend that you use the 64-bit version of ScalienDB.
  • The source release also builds under OSX and includes XCode project files.
  • The source code is available on Github.

Building

On Linux, download and extract the zip file or get the source from Github:

git clone git://github.com/scalien/scaliendb.git scaliendb

Then issue the following commands:

cd scaliendb
make
make pythonlib # optional
make javalib   # optional

On Windows, use the supplied Visual Studio project files.

Running

Launch the server with a specified configuration file:

./scaliendb test/control/single/scaliendb.conf

Once the server is up running, connect to http://localhost:8080/webadmin to see the Standard Management Console.

Don't forget to check out the Documentation.
